# GLib++
A proof-of-concept of reimplementing GLib as a C++ library.

## Implementation
This library is mostly based on the [C library](https://docs.gtk.org/glib/index.html), with some inspirations from [Vala's GLib implementation](https://valadoc.org/glib-2.0/index.htm).

GObject is planned, but I would like to get the core GLib stuff working first. Contributions are welcome, though.
